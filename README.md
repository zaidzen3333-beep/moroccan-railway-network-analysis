# Analyse et Optimisation du Réseau Ferroviaire Marocain

Ce projet utilise la théorie des graphes pour modéliser et analyser le réseau ferroviaire marocain.

## Contenu

- `Railway_Network_Analysis_v2_Robustness.ipynb`: Le notebook principal contenant l'analyse, la visualisation et l'étude de robustesse du réseau.
- `projet_reseau_maroc.csv`: Les données du réseau (Gares, Connexions, Distances, Durées, etc.).
- `requirements.txt`: Les dépendances Python nécessaires pour exécuter le notebook.

## Installation

1. Assurez-vous d'avoir Python installé.
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation

Ouvrez le fichier `Railway_Network_Analysis_v2_Robustness.ipynb` avec Jupyter Notebook ou JupyterLab pour explorer l'analyse.
